OBS Studio

Free and Open Source software for video recording and live streaming created by OBS PROJECT. Download and start streaming quickly and easily on Windows, Mac or Linux.

Features

  • Performance
  • Unlimited Scenes
  • Dock UI
  • Streamlined Settings
  • Audio Mixer

High performance real time video/audio capturing and mixing. Create scenes made up of multiple sources including window captures, images, text, browser windows, webcams, capture cards and more.
